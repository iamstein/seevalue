Tornado plot for adverse events

Overview

This module creates tornado plots to compare occurrence rates of multiple adverse events across two treatment arms. The module tests the null hypothesis

The occurrence rate of all adverse events is the same in both study arms.

Lineup Generation

The rows of the data table should corresponding to individual patients in the study. There should be should a column indicating to which study arm each patient was assigned (0 for control, 1 for treated). It should also contain a column for the occurrence (1) or non-occurrence (0) of each adverse event. The module creates null datasets by permuting the arm assignment column while keeping all of the other columns fixed.
